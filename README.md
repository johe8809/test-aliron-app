# test-aliron-app

## Instalación

1. Clonar el proyecto digitando el siguiente comando en la consola: 

```
git clone https://github.com/johe8809/test-aliron-app.git
```

2. Ejecutar el siguiente comando para instalar los paquetes que se encuentran en el package.json

```
yarn install
```

3. Luego ejecutar el siguiente comando por consola para arrancar la aplicación:


```
yarn start
```
